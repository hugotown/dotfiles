# Next.js App Router Implementation Guides

This skill provides comprehensive implementation guides for common Next.js App Router patterns and integrations.

## Available Guides

### Core Features
- **Forms and Mutations** - Server Actions, validation, optimistic updates
- **Database Integration** - Prisma, Drizzle, connection pooling strategies
- **Real-time Features** - WebSockets, SSE, polling patterns

### Data & Storage
- **File Uploads** - Handling uploads, cloud storage integration, streaming
- **Search** - Full-text search, third-party services, database search

### Integrations
- **Email** - Transactional emails, templates, provider integration
- **Payments** - Stripe integration, webhooks, subscriptions
- **Background Jobs** - Queue systems, cron jobs, serverless patterns

## Usage

Reference specific guide files for detailed implementation steps, code examples, and best practices for each feature area.

## Best Practices

All guides follow Next.js 14+ App Router conventions:
- Server Components by default
- Server Actions for mutations
- Proper error handling and validation
- Type safety with TypeScript
- Performance optimization
- Security considerations
